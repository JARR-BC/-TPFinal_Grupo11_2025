1. Semáforo para personas no videntes con alerta sonora

El sistema consiste en un semáforo automatizado que utiliza luces LED y señales sonoras para asistir a personas con discapacidad visual al cruzar la calle. Incluye un botón de petición de paso y un zumbador que emite sonidos diferenciados según el estado del semáforo.

4. Descripcion y funcionalidades.

   
Para este proyecto se simulará un paso peatonal para ciegos, adaptado a una avenida de 3 manos con un unico sentido de circulación, el cual tendrá 8 pasos peatonales (ida y vuelta) y una alarma sonora para cada paso peatonal, de esta manera la persona no vidente usará debidamente el paso mediante el sentido de la audicion.

Los semaforos seran realistas y continuos.
Los botones a cada lado de la calle colocarán el siguiente semaforo a rojo durante determinado tiempo.
La alarma sonora se activara una vez que el semaforo a cruzar esté en rojo y la misma pitará mas lento y constante para que con el paso de los segundos se vuelva rapida y menos constante dando la sensación de pérdida de tiempo para el peaton.
